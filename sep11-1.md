# Nested Loops in Java 

## 1. Row × Column
- **Outer loop** → controls rows 
- **Inner loop** → controls columns  



```java
int rows = 3;
int columns = 4;

for (int i = 1; i <= rows; i++) {       
    for (int j = 1; j <= columns; j++) { 
        System.out.print("* ");
    }
    System.out.println(); 
}
```
## 2. Row × Repetition (Triangle)

- **Outer loop** → controls rows 
- **Inner loop** → repeated actions inside outer loop

```java
for (int i = 1; i <= 4; i++) {
    for (int j = 1; j <= 4; j++) {
        System.out.print("*");
    }
    System.out.println();
}
```
```java
for (int i = 1; i <= 4; i++) {
    for (int j = 1; j <= 4; j++) {
        if (i==j){
            System.out.print("*");
        } else {
            System.out.print(" ");
        }
    }  
    System.out.println(); 
}
```
## 2. Row × Repetition (Triangle)

- **Outer loop** → controls rows 
- **Inner loop** → repeated actions inside outer loop

```java
for (int i = 1; i <= 4; i++) {
    for (int j = 1; j <= i; j++) {
        System.out.print("*");
    }

```

```java
for (int i = 1; i <= 4; i++) {
    for (int j = 4; j >= 1; j--) {
        System.out.print("*");
    }
    System.out.println();
}
```
